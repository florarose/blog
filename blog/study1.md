# 思考

只有愿望，没有具体的执行路线是不行。

**没有路线图的愿望 == 空想**

没有具体的路线，就意味着 **此刻的你** 和 **目标**之间，充满了未知，充满了变数。

**打个比方**：

目标： 去面试

准备工作：

   第一步： 找到面试地点

**第二步**： 规划路线，比如打开高德地图，看看路线图；可能一天要去好几个地方，先去哪，再去哪，早上去哪，下午去哪。

执行：

  有路线图，按路线图去。  

如果没有做准备工作的第二步，执行的时候，就会遇到各种问题,包括： 时间，距离等等。规划好了，就可以知道先去哪里，再去哪里，几点去。
